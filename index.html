How to record any of these as an MP4

Easiest path: open the ad in Chrome, then use Chrome's built-in recorder (DevTools → ⋮ → More tools → Recorder) or any screen-record extension. The ads auto-play on load and run to the end with a progress bar at the bottom.

Higher quality: use OBS, Loom, QuickTime (Mac), or ScreenRec. Set window size to match the target aspect ratio for clean export. The ads have a Replay button that appears at the end — hit it to re-run for retakes.

Pixel-perfect export: open the HTML in a browser, resize the window to exactly 1080×1920 (9:16), 1080×1080 (1:1), or 1920×1080 (16:9), then screen-record at native resolution. Trim the start/end in any video editor.
